# AI Portal Project

---

## 📖 프로젝트 개요

본 프로젝트는 **[ESTSOFT] 백엔드 개발자 양성과정 14회차**의  
**첫 번째 팀 프로젝트**입니다.

여러 AI 서비스를 한 곳에서 접근할 수 있는  
**포털 형태의 웹 페이지 UI 설계**를 목표로 하였으며,  
HTML / CSS / JavaScript 기반의 **프론트엔드 기술만을 사용하여 구현**하였습니다.

---

## 🎯 프로젝트 목표

- 프론트엔드 기반 검색 포털 사이트 구현  
- UI 중심의 페이지 구조 설계  
- 기본적인 기능 구현을 통한 웹 서비스 구조 이해

## 👥 팀 구성 및 역할

| 이름 | 담당 영역 |
|------|-------------------------------------------------------------------------------------------------------|

| 정상원 | 질문 결과 페이지, 로그인시 프로필, 공통 스타일 시트 aside 기능 java script, 고객지원 페이지, 도움말 페이지 |

---

## 🧩 주요 기능

- 메인 검색 기반 AI 포털 UI
- 질문 입력 및 결과 페이지 출력
- 로그인 / 회원가입 기능
- 사용자 정보 확인 및 수정
- LocalStorage 기반 데이터 관리
- 페이지별 역할 분리 및 구조화된 설계

---

## 🖥️ 구현 페이지

- 메인 페이지 (`index.html`)
- 로그인 페이지 (`login.html`)
- 회원가입 페이지 (`join.html`)
- 질문 결과 페이지 (`result.html`)
- 사용자 정보 페이지 (`editinfo.html`)
- 비밀번호 찾기 (`find.html`)
- 인증 관련 페이지 (`authentication.html`)
- 고객 지원 페이지 (`support.html`)
- 도움말 페이지 (`help.html`)
