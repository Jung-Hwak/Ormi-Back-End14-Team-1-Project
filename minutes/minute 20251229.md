# 1차 프로젝트 1팀 회의록

- 날짜: 2025년 12월 29일
- 참여자: 정확 (팀장), 정상원, 김영호

## 오전 회의 내용

> [!IMPORTANT]
>
> - `+ 새 채팅` 살짝 왼쪽으로 이동시키기 현재 오른쪽 벽에 너무 가까이 붙어있음
> - 아이디/ 비밀번호 찾기 페이지 연결하기
> - 메인 페이지 고객센터 도움말 연결하기
> - 메인페이지 새 채팅 버튼 만들기
> - 비밀번호 변경시 변경된 비밀 번호로 로그인 되게 만들기
> - 결과페이지의 결과 값을 동적 자바스크립트로 작동하기



## 작업 목표

> [!IMPORTANT]
>
> - 정확 : 아이디/비밀번호 찾기 연결, 메인페이지에 설정 창에서 도움말, 고객센터 연결하기 비밀번호 변경시 변경된 비밀번호로 로그인 되게 만들기
> - 정상원 : 결과값 동적 자바스크립트로 작성하기, 사이드바에 기능 추가하기
> - 김영호 : 누락된 부분 다듬기 들여쓰기 문서의 오타들 수정하기

## 작업 과정

> [!IMPORTANT]
>
> - 정확 : 메인 페이지에서 검색어를 입력하고 result.html로 넘어갔을 때, 검색어가 사라지거나 채팅방이 새로 생성되지 않는 문제 발생함 URL Query Parametere를 사용하여 값을 읽어와서 
>         채팅칸에 입력이 된것처럼 만듬, 검색에 대한 답이 그냥 바로 생성되는것 같아서 다른 ai검색 사이트에서 글자가 하나하나 생성되는것 같은방식을 찾기위해 검색을 해보니
>         타이핑 효과라는게 있어서 한글자씩 생성되는 것같은 효과를 적용함 
> - 정상원 : 코드 다듬기, 채팅방 이름 변경, 삭제 버튼 만들기(아직 못 함)
> - 김영호 : 누락됐던 자바 스크립트의 버튼을 다시 만들었고, 그 과정에서 몇몇 누락된 부분도 다시 작성했다. 덤으로 다른 파일들의 눈에 띄는 오타들을 수정하였다.


## 작업 회고

> [!IMPORTANT]
>
> - 정확 : 메인 페이지에서 검색한 키워드를 결과 페이지로 전달하고, 검색 결과의 값이 한글자씩 생성되는 듯한 효과를 어떻게 해야할지 웹 검색으로 찾아냄,
>         이용자가 검색을 할때 그 검색 결과 값을 프론트엔드에서 처리가 어떻게 해야되는지 고민이 되었음, 심플하게 검색 값과 그 결과값을 객체로 만들어서 저장해놓음
> - 정상원 : 채팅 내역을 수정해도 원하는 결과가 안나와서 힘들었다..
> - 김영호 : 다 한줄 알았던 자바 스크립트가 누락이 되어, 다시 원상복구를 하고 다른 파일들도 문제가 있는지 없는지 체크. 몇몇 파일들이 양식과 달라 다시 한번 재정리
>           몇몇 오타 수정
