# 1차 프로젝트 1팀 회의록

- 날짜: 2025년 12월 19일
- 참여자: 정확 (팀장), 정상원, 김영호

## 오전 회의 내용

> [!IMPORTANT]
>
> - 설정창 저장 취소 버튼 자바스크립트로 동작 기능 활성화 시키기
> - 메인 결과창에서 질문하는 chatForm에 엔터로 검색이되는 기능 연결시키기 (자바 스크립트)
> - 회원가입 페이지의 프레임과 스타일 시트 완성해보기
> - 로그인 페이지의 프레임만들기, 스타일 시트 작성해보기
> - 로고 이미지만들어서, 모든 페이지의 상단 가운데 부분에 이미지 적용시키고 이미지를 눌렀을때 메인 페이지로 넘어가는 기능



## 작업 목표

> [!IMPORTANT]
>
> - 정확 : 설정창 저장 취소 버튼 자바스크립트로 동작 기능 활성화 시키기, 로그인 페이지 프레임 짜보기, 스타일 시트 작업해보기,로고 이미지 적용시키기
> - 정상원 : 메인, 결과창에서 질문하는 chatForm에 엔터로 검색이되는 기능 연결시키기 (자바 스크립트), 로그인시 로그인버튼이 있던 것을 누르면 회원정보 관리등 나오는 프레임 만들기
> - 김영호 : 회원가입 페이지의 프레임과 스타일 시트 완성해보기, 본인 인증, 회원가입 페이지의 취소 버튼 클릭시 그전에 있던 페이지로 이동하는 기능구현


## 작업 과정

> [!IMPORTANT]
>
> - 정확 : ai를 이용해 로고를 만들어서 로고를 적용 하려고 했는데 로고에 체크무늬로 배경이 있어 스타일로 없에보려고 했지만 되지않아서 투명색 배경의 로고를 다시 만들어 스타일을
>         적용하니 제대로 보였고 로고가 들어갈자리에 span요소로 되어있던걸 div에 넣어서 앵커 요소로 href 속성을 추가해 메인 화면으로 넘어가지게 만들었음, 현재 로그인 화면의
>         HTML문서로 프레임을 만드는데 있어 id와 class등을 직관적인 영어이름을 적음
> - 정상원 : 메인페이지 결과페이지 모두 엔터키 적용 , 적용 후 생기는 오류 수정 , 로그인 후 로그인 프로필 창 만들기 
> - 김영호 : 첫번 째 HTML 문서 작성으로 기본적인 뼈대 만들기. 입력받을 항목(아이디, 비밀번호, 이름, 이메일)을 배열(Array)로 정의. 각 항목의 속성(id, label, type, placeholder)을
>           객체로 관리. input-id 필드 옆에 '중복 확인' 버튼을 동적으로 추가. 취소 버튼 클릭시 본인인증 페이지로 이동, 가입 완료 버튼을 클릭시 유효성 검사(아이디 중복, 비밀번호
>           확인 등)을 완료하고 로그인 페이지로 이동. 전반적인 스타일 시트를 외부 파일로 이동.


## 작업 회고

> [!IMPORTANT]
>
> - 정확 : 이미지를 만들때 배경색을 누끼를 따지않기 그냥 투명색 배경을 이용해야함 이미지를 넣을때 이 이미지가 콘텐츠를 대신하는건지 콘텐츠가 아닌지에 따른 배경 이미지로 넣을지 img의
>          src 속성으로 넣을지를 고민했던 결과 네이버 페이지의 작업 방식을 보고 따라해봄
> - 정상원 : 로그인 창 프로필파일을  만드는데 어떻게 시작 해야 될지 막막 했다.
> - 김영호 : 스타일 시트 분리, 회원가입창 완료. 아이디 중복, 비밀번호 확인, 이름, 이메일등 입력 받아 회원 가입 완료하게 함.
>           회원가입 페이지에서 취소를 누르면 이전 본인 인증 페이지로 이동, 가운데 로고를 클릭하면 메인 페이지로 이동하게 만듬.

## 내일 목표

> [!IMPORTANT]
>
> - 정확 - 로그인 페이지의 스타일 시트 완성하기, 로그인 페이지의 로그인 버튼을 누르면 로그인되는 기능, 회원가입 버튼을 누르면 회원가입 페이지로 넘어가기
> - 정상원 - 로그인창 프로필 파일 완벽하게 하기
> - 김영호 - 상단 네비게이션 바 통일성 있게 수정하기
>           비밀번호 확인란에서 비밀번호가 틀렸는지 아닌지 확인하게 만들기
>           전반적인 사용감 개선하기
